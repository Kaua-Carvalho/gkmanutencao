# GK Manutenção em Geral

Este projeto é um site desenvolvido para a **GK Manutenção em Geral**, com o objetivo de apresentar os serviços da empresa e facilitar o contato com clientes de forma rápida e profissional.

## 🚀 Funcionalidades

* Apresentação dos serviços oferecidos
* Acesso direto ao WhatsApp para orçamento
* Exibição de informações de contato
* Identidade visual personalizada (logo e cores da marca)
* Layout responsivo (mobile e desktop)

## 💻 Tecnologias utilizadas

* HTML5
* CSS3

## 🎯 Objetivo

Oferecer uma presença online simples e eficiente para a GK Manutenção em Geral, permitindo que clientes conheçam os serviços e entrem em contato de forma prática, rápida e organizada.
